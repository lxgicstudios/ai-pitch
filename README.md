Generate startup pitch deck content with AI.

## Install

```bash
npm install -g ai-pitch
```

## Usage

```bash
npx ai-pitch "AI-powered code review tool for teams"
npx ai-pitch "marketplace for freelance designers" -o deck.md
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
